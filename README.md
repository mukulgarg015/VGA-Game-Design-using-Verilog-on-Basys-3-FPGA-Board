# VGA-Game-Design-using-Verilog-on-Basys-3-FPGA-Board
In the designed game, the player has to reach the maze's end in a given time crossing all the hurdles. The player can also take certain powers available on the path. BROMs and different frequency clocks are extensively utilized.
